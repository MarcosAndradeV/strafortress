# Strafortress

## Idea

strafortress is a game writen in C++ using the nob build system and raylib graphics library.

Is a grid based game of resource collection, base building and tower defense.

The game map is made of 128x128, 256x256, 512x512 or 1024x1024 tiles, procedural generated, with biomes with diferent resources to be used in the expansion of the base.

Enemies spwan at ramdom cornes of the word and they attack any player based construction.

The player has no in game character and just manages this autonomus factory/base placing down constructions, machine, extractors, wall and turrents

Each 15-20 days a raid of enemies tries to attack uppon defeat a new section of the word is descovered given the player a  choice to expand or not, (enemies only spwan near players bases)
